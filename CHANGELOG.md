## v2.1.1

> Feb 22, 2019

- Now you can set a callback function on shortcut buttons, for example:

```javascript
const shortcuts = {
  Today: moment(), // Pure moment object
  Clear: /* moment + callback */ {
    moment: currentMomentObj, // REQUIRED
    callback: /* optional */ () => {
      this.setState({ cleared: true });
    }
  }
};
```
